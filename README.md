# Crimson Desert RenoDX Mod

**Crimson Desert RenoDX Mod** is a high-quality graphics enhancement preset for Crimson Desert that significantly improves visual fidelity, lighting, colors, and overall atmosphere using advanced RenoDX shaders.
Enjoy sharper textures, more realistic lighting, better shadows, enhanced foliage, and cinematic post-processing effects.

## ❗️ Troubleshooting

| Problem                    | Solution                                              |
|----------------------------|-------------------------------------------------------|
| Mod not working            | Make sure files are in the correct game folder        |
| Game crashes               | Disable other overlays (Discord, MSI Afterburner, etc)|
| Low FPS                    | Use lighter preset or lower shader quality            |
| Black screen / artifacts   | Update graphics drivers                               |

## ❓ FAQ

- **Is this mod safe?** Yes, it only modifies visual shaders.
- **Does it work in multiplayer?** Recommended for single-player only.
- **Can I disable it easily?** Yes, just delete or rename the RenoDX folder.
